##Dota2Recipe

![logo image][1]

Dota2图谱/秘籍, android app.

Dota2 游戏中英雄/物品资料合集软件, 全部数据离线.

###Developed By

* TUPUNCO - <tupunco@gmail.com>

###功能:
* 英雄列表, 英雄详细(属性, 统计信息, 技能, 推荐出装)

* 物品列表, 物品详细(属性, 合成所需物品列表, 可合成物品列表)
* 英雄物品收藏

###未完成功能:
* 搜索提供者
* 分享
